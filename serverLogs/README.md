# ServerLogsUI
## Server Logs UI is the Utility based on Dash which is Python framework for building web applications. It is built on top of Flask, Plotly.js, React and React Js. Code is written in python.

### Features of this utility are-
1. Status - Get the status and health of weblogic servers
2. Resolution of Issues - Find the root cause of weblogic server issue 
3. Secure - No data saving in local machine from Database
4. Interactive -It is highly interactive and user friendly web application.

## Installation steps to run in PC:
1. Install python interpreter(python-3.X.X.exe)
2. Install modules from requirements.txt
3. Set admin credentials in user.py
4. Run serverLogs.py using python interpreter and configure weblogic servers
